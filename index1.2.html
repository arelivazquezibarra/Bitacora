apuntes de la clase


La web funciona mediante links que nos llevan a otra pagina.

Un link puede ser cualquier cosa, una imagen, un video, un texto etc.

La sintaxis de un link es la siguiente
Apertura de estiqueta "a" atributo para definir el destino "href", destino "url", parte visble para el usuario "link", cierre de la etiqueta "/a".

Podemos usar una imagen como link simplemente poniendo "img" dentro de la etiqueta "a".

Existe el atributo target que especifica donde abrir el documento vinculado y puede tener los siguientes valores.
Self, blank, parent, top.





Resumen

Para la propiedad de los links se utiliza el atributo a, esto es fundamental saber ya que con estos atributos nuestra pagina se hace mas interactiva para el usuario, tambien para que sea mas facil de entender, tambien sabemos que se puede usar una imagen con un link o enlace.
