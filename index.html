ideas principales


En html se puede etiquetar varias propiedades algunas son

  • texto
  • imagenes

Estrucutura basica de un Archivo HTML:
Declaracion "HTML5"

Inicio del archivo "HTML"

Apertura de etiqueta "head"

Apertura de etiqueta "body"

Cierre de etiqueta "body"

Cierre de etiqueta "head"

Cierre del archivo "HTML"

apuntes de la clase


HTML es el componente mas basicos de la web define el significado y la estrucutura del contenido de la web
Generalmente se utilizan otras tecnologias para describir la aparencia o presentacion de una pagina web a la funcionalidad o comportamiento.

html utiliza los diples (<>) para etequitar texto, imagenes y otro contenido para mostrarlo en un navegador web, Las marcas html incluyen elementos especiales, debe tener un inicio y un final: "inicio - contenido - fin".

En la estrucutura basica de un Archivo html se declaran las variables, se establece el inicio del archivo, la apertura de la etiqueta, cierre de etiqueta, apertura de otra etiqueta, cierre de las etiquetas, cierre del archivo.

Resumen

En general el html es un lenguaje que define la estructura y contenido de una pagina web, donde se incluyen propiedades para dar forma y estetica y que sea agradable para el usuario, Sabemos que el html tiene una estructura concreta para que todas las propiedades sean leidas e implementadas de manera correcta, hay que tener muy en claro lo que se quiere realizar para poder implememtarlo en el html.
