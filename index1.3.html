apuntes de la clase


Las hojas de estilo en cascada es un lenguaje que define la apariencia de un documento escrito en un lenguaje de marcado.
Son un mecanismo simple para agregar estilo o diseño a los elementos de la pagina web creados con html, se le dara la apariencia que se desee utilizando css: formas, tamaños, margenes, tipos de letra, etc.

La hoja de estilo encrustada puede usarse en un documento unico, si la misma hoja de estilo se usa para multiples documentos entonces, seria mas apropieado una hoja de estilo externa.

La hoja de estilo en linea puede usarse mediante el atributo "style", si el estilo debe aplicarse a un unico elemento pero solo en cierto medio.

La hoja de estilo externa puede ser enlazada a un documento html mediante un elemento "link" de html.

Todos los estilos en una pagina se encadenan en una hoja de estilo "virtual"; Un estilo en linea tiene la maxima prioridad y anulara los valores internos y externos.





Resumen

Las hojas de estilo (css) es el lenguaje que ayuda con la apariencia de un documento, nos ayuda desde el acomodo de las cosas, hasta el color que tendran las propediades, tambien el css hace que nuestra pagina tenga todas las propiedades necesarias para que sea agradable e interactiva para nuestro usuario.
